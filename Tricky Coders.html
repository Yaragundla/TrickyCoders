Data Science

Data science is an inter-disciplinary field that uses scientific methods, processes, algorithms and systems to extract knowledge and insights from many structural and unstructured data.Data science is related to data mining, machine learning and big data.

Data scientist is one who practices the art of Data Science.

Life cycle of Data Science :

  1. Business Understanding..
  2. Data Collection..
  3. Data Preparation..
  4. Data Modelling..
  5. Interpreting Data..

Keynotes :

  • "Data science" became more widely used in the next few years: in 2002, the Committee on Data for Science and Technology launched Data Science Journal.. ...
  • Python is a programming language with simple syntax that is commonly used for data science.
  • R is a programming language that was designed for statisticians and data mining and is optimized for computation.
  • Apache Hadoop is a software framework that is used to process data over large distributed systems.
  • Dataiku is a collaborative data science software marketed for big data.
  • Databricks is a cloud platform for massive scale data engineering and collaborative data science.
